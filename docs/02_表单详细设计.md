### 2 基础数据模块

#### 2.1 部门表
| 字段 | 类型 | 必填 | 唯一 | 备注 |
|------|------|------|------|------|
| 部门ID | 文本 | 是 | 是 | 主键，如 DEP001 |
| 部门名称 | 文本 | 是 | 是 | 如“生产部”、“销售部” |

#### 2.2 职位表
| 字段 | 类型 | 必填 | 唯一 | 备注 |
|------|------|------|------|------|
| 职位ID | 文本 | 是 | 是 | 主键，如 POS001 |
| 职位名称 | 文本 | 是 | 否 | 如“生产经理”、“销售专员” |
| 所属部门 | 关联记录 | 是 | 否 | 关联部门表 |

#### 2.3 班组表
| 字段 | 类型 | 必填 | 唯一 | 备注 |
|------|------|------|------|------|
| 班组ID | 文本 | 是 | 是 | 主键，如 TEAM001 |
| 班组名称 | 文本 | 是 | 否 | 如“金工班”、“装配班” |
| 所属部门 | 关联记录 | 是 | 否 | 关联部门表（通常仅生产部有班组） |

#### 2.4 员工表
| 字段 | 类型 | 必填 | 唯一 | 备注 |
|------|------|------|------|------|
| 工号 | 文本 | 是 | 是 | 主键，如 EMP001 |
| 姓名 | 文本 | 是 | 否 | |
| 所属部门 | 关联记录 | 是 | 否 | 关联部门表（部门名称） |
| 所属班组 | 关联记录 | 否 | 否 | 关联班组表；仅当部门=生产部且职位=班组长/操作工时才显示 |
| 职位 | 关联记录 | 是 | 否 | 关联职位表（过滤条件：职位表.所属部门 = 员工表.所属部门） |
| 入职日期 | 日期 | 是 | 否 | 员工入职日期，用于业务发生时校验 |
| 状态 | 下拉框 | 是 | 否 | 在职/离职 |
| 联系电话 | 文本 | 否 | 否 | |
| 邮箱 | 文本 | 否 | 否 | |
| 简道云账号 | 文本 | 否 | 否 | |

#### 2.5 客户信息表
| 字段 | 类型 | 必填 | 唯一 | 备注 |
|------|------|------|------|------|
| 客户编码 | 文本 | 是 | 是 | 主键，如 C001 |
| 名称 | 文本 | 是 | 是 | 客户名称应唯一 |
| 类型 | 下拉框 | 是 | 否 | 国内/国外 |
| 国家 | 文本 | 否 | 否 | |
| 联系人 | 文本 | 否 | 否 | |
| 联系电话 | 文本 | 否 | 否 | |
| 邮箱 | 文本 | 否 | 否 | |
| 付款方式 | 下拉框 | 是 | 否 | 如“月结30天”、“款到发货”、“T/T”等 |
| 信用额度 | 数字 | 否 | 否 | |
| 合作状态 | 下拉框 | 是 | 否 | 合作中/潜在客户/已终止 |

#### 2.6 供应商信息表
| 字段 | 类型 | 必填 | 唯一 | 备注 |
|------|------|------|------|------|
| 供应商编码 | 文本 | 是 | 是 | 主键，如 S001 |
| 名称 | 文本 | 是 | 是 | 供应商名称应唯一 |
| 类型 | 下拉框 | 是 | 否 | 如“原材料”、“物流服务”、“外购件”等 |
| 主要物料 | 文本 | 否 | 否 | |
| 联系人 | 文本 | 否 | 否 | |
| 联系电话 | 文本 | 否 | 否 | |
| 邮箱 | 文本 | 否 | 否 | |
| 付款条件 | 下拉框 | 是 | 否 | 如“月结60天”、“预付30%”等 |
| 合作状态 | 下拉框 | 是 | 否 | 合作中/潜在/终止 |

#### 2.7 产品主数据表
| 字段 | 类型 | 必填 | 唯一 | 备注 |
|------|------|------|------|------|
| 产品编码 | 文本 | 是 | 是 | 主键，如 P001 |
| 名称 | 文本 | 是 | 是 | 产品名称应唯一 |
| 型号 | 文本 | 否 | 否 | |
| 类型 | 下拉框 | 是 | 否 | 整机/半成品/配件/外购件/原材料 |
| 规格参数 | 多行文本 | 否 | 否 | |
| 材质 | 文本 | 否 | 否 | |
| 重量 | 数字 | 否 | 否 | |
| 单位 | 下拉框 | 是 | 否 | 如“台”、“个”、“吨”、“公斤”、“米”等 |
| 版本 | 文本 | 是 | 否 | 产品版本号，如 V1.0 |
| 生效日期 | 日期 | 是 | 否 | |
| 状态 | 下拉框 | 是 | 否 | 启用/停用 |

#### 2.8 仓库仓位表
| 字段 | 类型 | 必填 | 唯一 | 备注 |
|------|------|------|------|------|
| 仓库编码 | 文本 | 是 | 否 | 如 WH01，与仓位组合唯一 |
| 名称 | 文本 | 是 | 否 | 如“原料仓”、“成品仓” |
| 类型 | 下拉框 | 是 | 否 | 原材料/成品/半成品 |
| 仓位 | 文本 | 是 | 否 | 如 A-01，与仓库编码组合唯一 |

#### 2.9 设备管理表
| 字段 | 类型 | 必填 | 唯一 | 备注 |
|------|------|------|------|------|
| 设备编码 | 文本 | 是 | 是 | 主键，如 E001 |
| 名称 | 文本 | 是 | 否 | 设备名称 |
| 型号 | 文本 | 否 | 否 | |
| 所在车间 | 文本 | 是 | 否 | |
| 状态 | 下拉框 | 是 | 否 | 运行中/维护中/停用 |
| 购入日期 | 日期 | 是 | 否 | 设备购入日期 |
| 上次保养日 | 日期 | 否 | 否 | 应晚于购入日期 |
