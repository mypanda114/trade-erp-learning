# 内外贸一体化管理系统（学习项目）

本项目为个人学习探索项目，旨在研究中小型制造业企业内外贸一体化管理系统的设计与实现。所有内容**仅用于学习研究目的**，不包含任何真实企业信息，不涉及商业用途。

## 项目说明

- 设计了一套覆盖销售、采购、生产、库存、外贸物流、财务等9大模块的通用型管理系统方案。
- 包含42个核心表单的详细字段定义，可直接用于简道云平台搭建。
- 提供完整的 Python 脚本，可自动生成模拟数据（300名员工、30条销售订单等），数据严格遵循业务逻辑（部门归属、在职状态、日期时序）。
- 所有脚本和文档均已本地 Git 管理，方便学习和回溯。

## 快速开始

### 1. 环境准备
- Python 3.6 或更高版本
- 虚拟环境（推荐）

```bash
# 在项目根目录创建虚拟环境
python -m venv .venv

# 激活虚拟环境（Windows PowerShell）
.\.venv\Scripts\Activate.ps1

# 安装依赖（使用 requirements.txt）
pip install -r requirements.txt
2. 生成模拟数据
bash
cd src
python generate_all.py
执行后，所有 Excel 文件将保存在 output/ 目录下，按模块分类，文件名已带数字前缀，可直接导入简道云。

3. 查看文档
详细的设计说明、表单字段定义、脚本规划及简道云搭建指南，请参阅 docs/ 文件夹下的 Markdown 文件。
